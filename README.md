</a><span style="font-size: 2em;">RarityGlow</span>

为不同稀有度的物品添加动态发光效果。

- **普通**：白色光晕
- **罕见**：黄色光晕
- **稀有**：青色光晕
- **史诗**：品红色光晕

支持自定义每种稀有度的开关与 RGB 颜色，配置实时生效。
